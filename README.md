# Flutter Overlay - Parte 2

Projeto em Flutter para demonstrar a criação e uso de Overlays personalizados.

- Version: Flutter 3.10.4

### Run
1. Clone o repositório
2. Execute `flutter pub get`
3. Execute `flutter run`
